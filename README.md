# Abdelbasset Lettreuch - Portfolio

A modern, animated portfolio website built with React, TypeScript, and Framer Motion.

![Portfolio Preview](https://img.shields.io/badge/React-19.2-blue?logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5.8-blue?logo=typescript)
![Vite](https://img.shields.io/badge/Vite-6.2-purple?logo=vite)

## Features

- **Modern UI/UX** - Clean, responsive design with smooth animations
- **Framer Motion** - Dynamic animations and transitions throughout
- **TypeScript** - Full type safety for better development experience
- **Vite** - Lightning-fast development and build times
- **Responsive** - Optimized for all screen sizes
- **Contact Form** - Integrated with Web3Forms for email delivery

## Tech Stack

- **Frontend:** React 19, TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Build Tool:** Vite

## Sections

- **Hero** - Animated introduction with profile image
- **About** - Personal background and experience
- **Skills** - Technical skills showcase
- **Projects** - Portfolio of work with live demos
- **Architecture** - Technical approach and methodologies
- **Contact** - Working contact form

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/basstop798/Abdelbasset-Lettreuch-portfolio.git

# Navigate to project directory
cd Abdelbasset-Lettreuch-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
npm run preview
```
